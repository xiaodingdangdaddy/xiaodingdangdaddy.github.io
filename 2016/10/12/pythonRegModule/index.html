python正则表达式re模块

2016-10-12

简介

Python re模块使得python语言拥有全部的正则表达式功能
compile函数根据一个模式字符串和可选的标志参数生成一个正则表达式对象。该对象拥有一系列方法拥有正则表达式匹配和替换。
re模块也提供了与上面方法功能完全一致的函数，这些函数使用一个模式字符串作为他们的第一个参数。

re.match函数

re.match尝试从字符串的起始位置匹配一个模式，如果没有匹配成功，返回一个None

re.search方法

re.search(pattern, string, flags=0)
re.search与match的表达式和参数类似，但是二者有很大的不同：
re.match只匹配字符串的开始，如果字符串开始不符合正则表达式，则匹配失败，函数返回None；而re.search匹配整个字符串，直到找到一个匹配。
在下面这个例子就可以很清楚的看出二者的不同

检索和替换

re.sub用于替换字符串中的匹配项
re.sub(pattern, repl, string, max=0)
pattern 表示正则表达式
repl 表示string中被pattern匹配到后替换成的内容
string 表示被匹配项
max 缺省为0.表示替换所有匹配项。表示匹配后替换的最大次数，比如max设置为3，表示在匹配到4个的时候只替换3个。
